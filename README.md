# Albania Brain Drain Analysis

## Project Overview

This project analyzes Albania’s brain drain using public data from the World Bank, Eurostat, INSTAT, and migration-related reports. The goal is to study how migration, population decline, unemployment, GDP growth, residence permits, and demographic structure are connected.

## Research Question

What economic and demographic factors are connected to migration and brain drain from Albania?
