# Methodology Plan

## Research Design

This project uses a quantitative descriptive and exploratory research design. It combines public macroeconomic, demographic, migration, and residence permit datasets to study Albania's migration pressure and brain drain.

## Main Steps

1. Understand each data source.
2. Clean and transform raw datasets.
3. Build a master dataset by year.
4. Analyze population, migration, economy, and residence permits.
5. Test statistical relationships using correlation and hypothesis tests.
6. Forecast selected indicators to 2035.
7. Present findings in a Power BI dashboard and research report.

## Important Warnings

- The project should not fabricate results.
- Calculated findings must come from the datasets.
- Limitations must be explained clearly.
- Eurostat proxy variables must be interpreted carefully.
